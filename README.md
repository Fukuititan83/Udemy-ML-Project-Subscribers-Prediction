# Udemy-ML-Project-Subscribers-Prediction
End-to-end Machine Learning project on Udemy finance courses dataset. Performed data cleaning, feature engineering, and built a Random Forest model to predict course subscribers. Includes EDA, visualization, and insights on factors affecting course success.
# 🚀 Finance & Accounting Courses Analysis (ML Project)

## 📌 Project Overview
An end-to-end Machine Learning project that analyzes 13K+ Udemy finance and accounting courses to uncover insights and predict course success. This project focuses on identifying key factors that influence course popularity and subscriber growth.

---

## 🎯 Objectives
✔️ Predict number of subscribers using ML  
✔️ Analyze impact of ratings, reviews, and pricing  
✔️ Discover patterns affecting course performance  

---

## 🛠️ Tech Stack
🐍 Python  
📊 Pandas, NumPy  
📈 Matplotlib, Seaborn  
🤖 Scikit-learn  
📓 Jupyter Notebook  

---

## 📂 Dataset
- 13,000+ Udemy courses  
- Features include:
  - Course Title  
  - Price (Original & Discounted)  
  - Ratings & Reviews  
  - Subscribers  
  - Course Content Details  

---

## ⚙️ Project Workflow
🔹 Data Cleaning & Preprocessing  
🔹 Handling Missing Values  
🔹 Feature Engineering  
🔹 Model Building (Random Forest)  
🔹 Model Evaluation (MSE, R²)  
🔹 Data Visualization  

---

## 🤖 Machine Learning Model
👉 Random Forest Regressor used for predicting course subscribers  

---

## 📈 Key Insights
📌 Courses with higher ratings & reviews attract more subscribers  
📌 Pricing strategy significantly impacts course success  
📌 Data-driven decisions improve prediction accuracy
